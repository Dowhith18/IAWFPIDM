3.7 POWER SUPPLY STAGES
EMS ECU has 4 supply stages
•  Power supply stage A
•  Power supply stage B
•  Power supply stage C
•  Power supply stage D
Power supply stage A
•  Throttle position sensor 1 & 2
•  Oil pressure sensor
•  Rail Pressure sensor
•  Intake camshaft sensor
•  Accelerator pedal sensor (APP1)
•  Clutch position sensor
•  AC pressure sensor
Power supply stage B
•  Gear Neutral sensor
•  Exhaust camshaft sensor
•  Boost Pressure sensor
•  Manifold pressure sensor
•  Crank position sensor
Power supply stage C
•  Accelerator pedal sensor (APP2)
Power supply stage D
•  eVGT position sensor
